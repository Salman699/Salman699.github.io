Format Tanggal Di JavaScript

Sebelum negara api menyerang, programmer harus mengeluarkan sedikit effort lebih untuk mengubah sebuah string menjadi format tanggal, tapi sekarang jadi lebih mudah lho. Ketika saya menerima sebuah string yang bentuknya sebuah tanggal dari API, saya sedikit kesulitan untuk memformat data tersebut. Saya mendapatkan saran dari teman untuk menginstall sebuah library, namanya date-fns. Tapi saya pikir-pikir, bisa gak ya, saya format pake kode saya tanpa harus install sebuah library?.

Ternyata ada sebuah method dari JavaScript yang ngebantu saya buat format tanggal. Method ini me-return sebuah string dari tanggal yang di format, dan sesuai dengan konfigurasi yang saya atur. Karena saya ingin membuat function ini reusable, saya akan membuat sebuah function yang nantinya akan memformat sebuah string ke format tanggal, misalnya.

Agar lebih terskala, saya menambahkan 2 tambahan parameter, yakni

  • locale Untuk menentukan region apa yang digunakan untuk memformat tanggal tersebut, contohnya dari function yang saya buat sebelumnya yaitu memformat tanggal ke dalam format Indonesia.
  • config Untuk mengatur format dari tanggal tersebut, yang menerima sebuah object, object ini akan berisi property year, month, dan day.
