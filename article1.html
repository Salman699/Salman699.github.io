Perjalanan Yang Menarik, Memulai Kembali Dan Menjadi Seorang Developer - Rekap Tahun 2021.

Introspeksi, sebuah kesadaran diri

Mengawali Tahun 2021 merupakan hal yang berat juga banyak hal-hal yang terjadi. Pertama, sebelum saya terjun ke dunia pemrograman, saya adalah seorang yang senang bermain game MMORPG dan MOBA, aktivitas setiap hari saya hanya sekedar bermain game didepan komputer, terkadang saya tidak tidur seharian hanya dengan bermain game saja. Tapi kala itu saya sadar diri dengan keadaan keluarga saya, yap.. saya terlahir di keluarga yang biasa saja, meskipun saya mengambil Jurusan Informatika, tapi waktu itu pandemi muncul dan aktivitas kuliah terhenti, karena hal ini lah, kenapa saya senang bermain game MMORPG dan game lain seharian karena tidak ada aktivitas kuliah yang padat. Pada bulan Februari 2021, saya memutuskan untuk berhenti bermain game, dan mulai melakukan esksplorasi pada bidang Web Development, karena seperti yang saya bilang sebelumnya, saya masuk Jurusan Informatika.

Menjadi Domba Tersesat

Akhir semester sudah dekat, saya masih mengikuti tutorial tanpa arah, dimana saya belajar dasar dari HTML di W3 Schools dan tidak terurut, bahkan waktu itu saya sangat kesulitan untuk membuat sebuah Hidden Nav Menu. Tidak sampai disitu saja, saya melanjutkan pembelajaran ke CSS, dan disini lah saya jatuh cinta di Frontend Web Development, karena jujur saja, sebetulnya saya ini sangat senang untuk mendesain sesuatu, tapi saya gak bisa pake tools seperti Adobe Photoshop, tapi saat saya membuat beberapa baris kode menggunakan CSS, tampilan dari halaman web sesuai dengan apa yang saya pikirkan.

"Kamu yang sedang membaca mungkin akan terheran-heran kenapa saya lebih suka mendesain sesuatu dengan CSS daripada menggunakan Software Desain, ya seperti pada headline diatas, Menjadi Domba Tersesat."

Cahaya Baru, Sebuah Petunjuk

Setelah kurang lebih 2 Bulan menjadi domba tersesat, saya pun menemukan sebuah channel YouTube yang sangat keren, yaitu Web Programming UNPAS.Di channel YouTube tersebut saya banyak belajar, ada playlist untuk Frontend Developer juga, jadi saya memulai lagi dari awal Playlist, sayapun mengikuti roadmap Frontend Developer, dimulai dari materi HTML, CSS dan juga Javascript.

Penutup

Menutup tulisan kali ini segitu dulu kurang lebihnya mohon maaf. Memang tulisannya gak begitu bagus, tapi saya harap kamu dapat memahami apa yang saya tulis, dan tau bagaimana perjalanan saya di Tahun 2021. Untuk kamu yang baru saja terjun di dunia Informatika, semangat! saya yakin sama kamu, kamu pasti bisa, tetap konsisten belajar and good luck!
